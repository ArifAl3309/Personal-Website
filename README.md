## ğŸ”§ Kustomisasi

### Mengubah Warna

Edit file `app.style.css` pada bagian color properties:

```css
/* Contoh: Mengubah warna header */
header {
    background-color: #607D8B; /* Ganti dengan warna pilihan Anda */
}
```

### Mengubah Font

Edit bagian Google Fonts di `<head>` HTML:

```html
<link href="https://fonts.googleapis.com/css2?family=NamaFont:wght@300..700&display=swap" rel="stylesheet">
```

### Menambah Konten

Tambahkan section baru di dalam tag `<article>`:

```html
<section id="new-section">
    <header>
        <h3>Judul Baru</h3>
    </header>
    <p>Konten Anda di sini...</p>
</section>
```
